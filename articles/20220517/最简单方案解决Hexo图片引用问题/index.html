最简单方案解决Markdown图片引用问题

发表于 2022-05-17 | 分类于 techStack | | 阅读次数: |

相信大家写markdown最大的困扰就是图片的嵌入，最挫的办法就是截屏+ 保存到本地 + 上传到图床 + 链接插入到markdown文档中。

每次这么搞一次我都要崩溃一次，一直在寻找一个比较方便和快速的图片直接粘贴到文档中的方法。

本文希望解决的问题

  • 可以直接截图（任意截屏工具），然后“Ctrl + V”自动将图片粘贴到markdown文档中；
  • 图片保存在本地，hexo d时自动将图片同步到github上；
  • 有一个比较好用的MarkDown编辑工具。

MarkDown编辑工具建议

  • 强烈推荐微软的VSCode编辑器。在使用vscode之前，尝试过hexo-admin/markdown.md/markdownPad2/Tyope/小书匠/有道…… 都存在各种各样的问题，最终还是留在了vscode。说说选择它的理由：
  • 免费；
  • 基础功能具备，且可通过插件进行功能扩展；
