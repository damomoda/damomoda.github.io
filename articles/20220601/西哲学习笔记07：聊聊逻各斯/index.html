西哲学习笔记07：聊聊“逻各斯”

发表于 2022-06-01 | 分类于 bookNotes | | 阅读次数: |

哲学特别是古希腊哲学就好像“阿格雷”的linux命令行界面，再优美的内涵也会被极端不友好的用户交互拖累。反正都是恶心，今天就聊聊我个人觉得第二恶心的“逻各斯”。

现代人往往把逻各斯等同于理性或逻辑，但在古希腊逻各斯大概有至少11种意思，而且逻各斯在古希腊文中也并不是专业的哲学概念，而是日常生活经常会使用的含义极端丰富的多义词。

其中第一个值得说的就是：话语。话语可以为世间万事万物命名，可以收集、聚拢和分辨。话语用命名的方式让世界变得清晰可分辨。话语可以突破人肉身的限制，用言说去采集世间万物，去触碰最抽象的爱情、原子、上帝……把上下五千年、纵横八万里的万事万物都采集、拢集、汇聚到此时此刻此地。

另外一个值得说的就是：尺度。赫拉克利特说：“这个世界……过去、现在、未来永远都是一团永恒的活火，在一定的分寸（尺度/逻各斯）上燃烧。”这里的“分寸”是复数形态，它强调的不是单一的尺度，而是多种尺度。只有借助于多种尺度，善于掌握分寸感，世界才会向我们呈现出它的丰富多样性。
