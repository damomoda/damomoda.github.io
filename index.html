西哲学习笔记18：柏拉图的政治观

发表于 2022-06-10 | 分类于 bookNotes | | 阅读次数:

鲁迅在《呐喊》自序里面写过一段话：……假如一间铁屋子，是绝无窗户而万难破毁的，里面有许多熟睡的人们，不久都要闷死了，然而是从昏睡入死灭，并不感到就死的悲哀。现在你大嚷起来，惊起了较为清醒的几个人，使这不幸的少数者来受无可挽救的临终的苦楚，你倒以为对得起他们么？……

这首先是一个人生哲学问题：你是愿意在昏睡中死灭，还是在清醒中死灭？其次，这也是一个政治哲学问题：你希望生活在一个没有自由但从子宫到坟墓都有保障的社会，还是生活在一个有自由也要承担很多个体责任的社会？

我们回到古希腊，两个我们熟悉的典型城邦：雅典和斯巴达。这两个城邦就是在实践两条完全不同的政治秩序。雅典实行的是民主制，而斯巴达则实行的贵足军事专制。雅典相信光明总是会带来温暖，斯巴达则相信保持某种无知是一种福分。从现实政治的效果来看，雅典创造了辉煌的文明，但是它在人心秩序上的混乱和政治秩序的不稳定也是一目了然；相比之下，斯巴达却是万众一心，保持了长达400年的政治稳定。两相比较，对于内乱频仍、渴望稳定的希腊人来说，斯巴达政体显然更有吸引力。

如果你生活在当时的希腊，你更愿意在哪个城邦生活？很显然柏拉图更倾向于生活在斯巴达，柏拉图把斯巴达的制度视为社会政治的理想状态，他更倾向于精英统治、秩序稳定和政治廉洁，他从斯巴达那里汲取了很多的思想资源。

柏拉图（Plato）出生于公元前427年，那一年第88届古代奥林匹克运动会开幕了。60岁之前的柏拉图在哲学史上一片空白。不过根据第欧根尼的《名哲言行录》记载：柏拉图出身望族，族谱可以向上追溯到雅典历史上的一位君主，母亲可以追溯梭伦（梭伦曾经把自己的族谱追溯到了海神波塞冬。也就是说，柏拉图是海神波塞冬的后裔）。

据说，有一天苏格拉底梦见一只小天鹅飞来停在他的膝盖上，发出嘹亮美妙的鸣声后一飞冲天。次日他就认识了柏拉图，于是苏格拉底就想当然的把柏拉图视为那只梦中的小天鹅。苏格拉底被处死的时候柏拉图年仅28岁，虽然柏拉图只在苏格拉底身边陪伴了8年，但他却用了一辈子的时间记述苏格拉底的对话。柏拉图在一封与友人的信中这样写道：“过去和将来都不会有柏拉图写的著作，现在以他署名的作品都属于苏格拉底、被美化与恢复了本来面目的苏格拉底。”是不是有点“青藤门下狗”的意思？

苏格拉底和柏拉图是雅典城邦贡献给世人的仅有的两个伟大的哲学家。柏拉图在青年时期对政治充满了热情，但是苏格拉底之死让他对现实政治心灰意冷，对民主制更是彻底丧失信心。苏格拉底的死，可以说是柏拉图失去天真的关键时刻，从此他离开雅典，自我放逐，四处游历，直到12年后才重返雅典。这期间柏拉图一直在思考政治问题，并且逐渐形成了他最为核心的政治判断：

“事实上，我被逼得相信，社会或个人找到正义的唯一希望在真正的哲学，以及，除非真正的哲学家掌握政治权力，或政客拜奇迹所赐变成真正的哲学家，否则人类永无宁日。”

还记得文首鲁迅《呐喊》的自序吗？其实，那段话的后面紧接着还有一句话：

然而几个人既然起来，你不能说决没有毁坏这铁屋的希望。

西哲学习笔记17：雅典民主制度和苏格拉底之死

发表于 2022-06-08 | 分类于 bookNotes | | 阅读次数:

雅典实行是直接民主制，人民既是统治者又是被统治者，没有任何中介和代表。全民通过“陶片放逐法、抽签”等方式进行重大政治事务的决策。

陶片放逐法，就是雅典公民开大会的时候在陶片上写下不喜欢的公众人物的名字，排名第一的就要被赶出雅典。这个制度的设计初衷是为了限制和约束因为名望和声誉过于隆重以至于可能威胁到城邦安全的卓越人士，防止僭主制的复辟。“陶片放逐法”的弊端也是一目了然的。雅典公民的教育层次参差不齐，有不少人干脆就是文盲。有一次，雅典著名的政治家阿里斯提德在投票的时候，就被一个公民要求代写某人的名字，结果阿里斯提德一听：这不就是我自己的名字吗？于是他好奇地问道：你为什么要放逐阿里斯提德呢？这个公民回道：不为什么，我其实并不认识他，但是因为总是听人赞美他是“公正之士”，我实在听烦了，不如就把他放逐算了。所以说“陶片放逐法”的优点是能够约束官员权力，防止僭主出现；缺点则是民意难测，而且极易成为党争的工具。据说当时甚至出现过先写上政敌的名字，然后批量生产陶片，分发给民众的情形。

抽签制，就是除了像将军和司库这类特别需要专门技能的官员是通过选举产生的，绝大部分的公职都是通过抽签的方式任命的，抽中的人就可以从事这个公职。抽签从表面上最大程度地实现了平等这个价值。坏处也是一目了然的。所谓术业有专攻，如果靠抽签来决定谁担任哪个职位，肯定会出现“德不配位”的情况。此外，抽签制看似强调平等，其实却是在伤害平等。就像亚里士多德所说的那样，以不平等的方式对待平等之人是为不平等，以平等的方式对待不平等的人同样也是不平等。

既然雅典的民主制度有这样那样的问题，那民主制是否就是个坏东西呢？

西哲学习笔记14：苏格拉底：无人有意作恶

发表于 2022-06-06 | 分类于 bookNotes | | 阅读次数:

你一定听说过身边的朋友说过类似的话：“戒烟是天底下最容易的事情，我都戒过许多回了。”、“减肥是天底下最容易的事情，我都减过许多回了”……

在戒烟和减肥的问题上人们之所以屡战屡败，通常的解释是“意志薄弱”或“不能自制”（akrasia）。但从哲学的角度说，“应该”蕴含着“能够”。打个比方，你“应该”珍惜生命、远离烟草，这个要求之所以是合理的，是因为你“能够”做到戒掉香烟。反过来说，对于不“能够”做的事情，我们就不会说“应该”。如果让苏格拉底来分析这个问题，我想他会说：“这不是因为你忍不住，而是因为你并没有真正认识到那是罪恶的，你的错误在于缺乏知识，没有看见真正的善。”
